JFFM - Just For Fun Menu

Tutorial:
```
1.Drücke auf JFFM.bat

2.gebe ein der vorgegebenen commands ein

3.Spaß haben :)
```

Fetures:

```
exit - Beenden
dz - datum/zeit (erstellt eine datei mit datum und uhrzeit)
hallo - sagt hallo
fun - Spas!
quest - du kannst eine aufgabe fuer dich erstellen
aufgabe - hier kannst du sehen welche aufgaben du hast
remove - deine aufgabe wird entfernt
shutdown - tut den pc in 10 sek herunterfahren
crash - öffnet ganz viele anwendungen und bringt den pc zum absturz, ich übernehme keine haftung ihr werdet 2 mal gewarnt
```
